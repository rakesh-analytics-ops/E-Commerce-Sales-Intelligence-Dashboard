# E-Commerce Sales Intelligence Dashboard

## Project Overview
Developed a 5-page Power BI dashboard to analyze sales performance, profitability, customer behavior, product intelligence, and regional trends.

## Dashboard Pages
1. Executive Overview
2. Category Intelligence
3. Customer Intelligence
4. Product Intelligence
5. Regional Intelligence

## Key KPIs
- Total Sales: 49M
- Total Profit: 7M
- Total Orders: 15K
- Quantity Sold: 45K

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV
